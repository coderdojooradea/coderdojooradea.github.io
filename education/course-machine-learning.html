Objectives

Hey there! From 24th of February Coder Dojo Oradea is starting a fresh new session on Machine Learning. This one is sponsored by Fundatia Comunitarea Oradea through Fondul Stiintescu. If you want to participate, please make sure you read the details below.

What is Machine Learning?

  • Machine learning is a field of computer science that gives computers the ability to learn without being explicitly programmed.

What do you need in order to register?

  • As you might have noticed, english is required! All online courses will be in english!
  • Know to program in at least one programming languange(C/C++, Python, Octave, MATLAB)
  • And the most important one, the will to learn new things!
  • Be atmost 17 years old :)

What do we focus on?

  • Our main focus will be building an Identity Recognition system using Computer Vision and Raspberry PI
  • You'll learn to work with an open-source machine learning framework - Tensorflow - Object Detection
  • You'll learn to program a Raspberry PI and a camera module for live detection

What reseach will we do?

  • We will first try to identify the face and name of a person from an image using manual labeling of data
  • We will then try to automatically draw the contour of a person's face using a trained model after which we'll just attach the label - should be for a faster labelling of data
  • We'll also look on how we could automatically take labelled images from Facebook (thanks to everyone who tagged a friend)
  • If we have time, try out a speach to text labeling technique using two trained models: speach to text + face recognition

Do I need to bring something with me?

  • Yes! Please don't forget your laptop as you'll use it during the sessions!
  • Bring also a mouse, it makes for a better experience as the touch pad is not that great when you're writting code!
